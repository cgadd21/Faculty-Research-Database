Once the program is open, a console menu will be shown
the menu will prompt the user to login, create an account,
or if the user is a guest they can access their own menu.

Login will login either student or faculty.
Sign Up will create a new user and then login.
Guest will login or creaate new guest account then login.
Exit will exit the program.

postLoginMenu is the second set of menu options.
Update user infomation will update the user.
Add interest will add interest.
Delete user will delete the user.
Search interest will search interests depending on the usr type.
Logout will logout the user and exit the program.
